Note di implementazione

1. Media Player
Tile a 2 colonne. Artwork placeholder con linear-gradient. Controlli con caratteri Unicode (⏮ ▮▮ ⏭).
2. Camera
Tile a 2 colonne con aspect-ratio: 16/9. Badge LIVE con @keyframes badge-live-blink. Timeline in position: absolute; bottom: 0.
3. Scenario / Scena — Ripple
Il ripple e un <span class="ripple"> aggiunto dinamicamente via JS. La classe .scenario-done viene applicata per 1.5s dopo il tap. Nessun modulo ES — tutto in IIFE / funzioni globali.
Vincoli iOS 12 Safari rispettati in tutto il mockup (v1.5)
Nessun optional chaining ?.. Nessun nullish coalescing ??. Nessun gap nei flexbox container interattivi (rimpiazzato da margin). Nessun dialog nativo. Nessun ES module / import / export. Nessun Promise.allSettled. Nessun structuredClone. Nessun .at(). CSS Grid usato liberamente (supportato iOS 10.3+). CSS Custom Properties OK (iOS 9.3+). animateMotion + mpath supportati su WebKit iOS 12. -webkit-appearance: none su tutti gli slider. Bottom sheet position: fixed; bottom: 0; left: 0; right: 0; z-index: 200. touch-action: manipulation su tutti i bottoni. -webkit-tap-highlight-color: transparent globale.
